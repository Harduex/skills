# Agent Skills

A catalog of reusable [Agent Skills](https://agentskills.io) for AI-assisted software development. Each folder contains a self-contained skill that can be pulled into any project.

## Usage

Install skills using any compatible AI agent package manager, or copy individual skill folders directly into your project.

Each skill follows the [Agent Skills specification](https://agentskills.io/specification) — browse the folders to see what's available.

## Creating skills

See the `write-a-skill/` folder for the authoring guide.
